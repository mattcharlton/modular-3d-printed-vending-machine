# modular-3d-printed-vending-machine


Arduino code for the 3D printed modular vending machine by Martin - found here https://makerworld.com/en/models/2380835-full-size-modular-vending-machine?from=search#profileId-2607127

This code is not official and has been thrown together by me with the help of Openclaw/ChatGPT with some of the trickier logic.  

It's still a work in progress and you'll need to bear pin connectivity in mind and make sure you're wired up properly.

The code isn't yet final, I still have some LED light strips to add and another relay to tie these to vend logic with putting a credit in. 

Please go to Makerworld and boost this amazing model.

Hope this helps people to get started - feel free to borrow/use bits for your own machines.  Good luck!
